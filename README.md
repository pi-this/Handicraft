# Handicraft 1.4

### What's new?

- Bug Fixes
  - Fixed the offsetting of a drawing after it was saved.
  - Fixed the issue of drawing strokes disappearing, fading, going bigger or smaller, and the issue of the stokes/dots moving around on the canvas after the project has been saved.
  - Added the xold and yold to the program where they were missing.
  - Fixed the image grab not defined error.
  - Changed the undo function to remove an entire stroke at a time instead of every dot in that line.
  - Added the sync_pil_image function for saving undo and redo actions into the file when it is saved as a PNG file. This fixed the bug of the program saving process not syncing in the undoing after the project was saved as an image.
  - Fixed spelling mistake
  - Modified the b1down and motion functions in order to fix multiple specific strokes and saving problems.
  - Fixed multiple issues with the pixel tool, the pencil tool, and the paintbrush tool. They are now compatible with saving and don't have issues related to that.
  - The pencil tool no longer creates crazy hair stokes, it now works very similar to the way the paintbrush works.
- Improved the Software's design
  - Rearranged the drop down menus
    - Turned Save As into Export (.png)
    - Turned Open As into Import (.png)
    - Removed the old save button
    - Remove the old open button
    - Removed the open sticker button
    - Removed the screenshot button
    - Changed the arrangement of the drop down items under "file"
    - Added the accelerator to the file menu, edit menu, view, and tools for better shortcut visualization.
    - Added 10 new shortcuts to the program. (7 for tools, CTRL+I for import, CTRL+E for export, and CTRL+Y for redo.
    - Added the "Open Recent button under the "file" drop down for opening recent projects.
  - Changed the images for the button tool bar to images created directly from Handicraft.
  - New Handicraft 1.4 intro.png image.
  - Changed the button tool bar to white instead of light gray
- Created a new way to save and open files with the .hcraft extension.
- Changed the "fill all" tool to the "fill" tool. It no longer fills the background, but it now fills any desired enclosed area that is clicked using the tool.
- Added the color picker/eye dropper tool for selecting a color that is already on the canvas.
- The initial tool is no longer the touch tool, it is now the pencil tool. This was changed for quicker ready to draw functionality for the application.
- The initial size of a tool is now 5 instead of 1.
- Added the ability to redo by creating a new redo function and a list specific to redoing.
- imported pickle for the .hcraft file extension
